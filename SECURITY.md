# Security and Publishing Notes

This repository is intended to remain public as a marketing site only.

## Allowed in this repo
- public website copy
- public design assets
- static site files for jmmtechnologies.app

## Do not commit here
- private product code
- client data
- internal notes or roadmaps
- credentials, API keys, tokens, or secrets
- infrastructure config meant to stay private
- unpublished project details

## Recommended practice
Use separate private repositories for all internal products, experiments, and operational code.
